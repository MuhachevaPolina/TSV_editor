# 1687-tsv-editor-pmukhacheva

Репозиторий задания "Табличный редактор TSV"

Выполнила: Мухачева Полина

## Описание

Приложение предназначено для просмотра и редактирования файлов формата TSV в виде таблицы.

## Запуск

### Вручную

1. Удалить директорию build/, если она уже существует

```bash
rm -rf build
```

2. Создать build/ и запустить из неё сборку проекта
```bash
mkdir build
cd build
cmake ..
make
```
Запустить приложение
```bash
./app
```

Запуск тестов

После сборки из директории build/ выполнить команду

```bash
./tests
```
